# パスカルの三角形を使って、ハンディキャップを計算しよう

**(2024-09-18 05:16) この記事は意味がなくなった**  


**先手ｍ本先取、後手ｎ本先取** といったルールを作ったとき、先手と後手の勝率の比がどれぐらいになるのかは、パスカルの三角形を眺めれば計算できます  


# 先手１本先取／後手１本先取制

```
+---+
| 1 |   先手１回勝ち
+---+

後手
１回勝ち
```

先手と後手の勝率の比　１：１　（先手勝率５０％）  

※先手勝率＝先手の勝率の比／（先手の勝率の比＋後手の勝率の比）  


# 先手２本先取／後手１本先取制

```
+---+   +----+
| 1 +---+  1 |   先手１回勝ち
+---+   +----+

後手　　　後手
１回勝ち　１回勝ち
```

先手と後手の勝率の比　１：２　（先手勝率３３％）
